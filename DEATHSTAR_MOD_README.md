# Death Star Mod

A NeoForge mod for Minecraft 1.21.1 that adds a Death Star block capable of creating massive 1000x1000 block explosions!

## Features

- **Death Star Block**: A powerful black obsidian-like block
- **Massive Explosions**: Right-click the Death Star block to trigger a devastating 1000x1000 area explosion
- **Destruction**: All blocks in the explosion area are destroyed
- **Effects**: Sound and visual explosion effects

## How to Use

1. **Build Requirements**: Java 21 or higher
2. **Build the mod**: Run `./gradlew build` (on Windows: `gradlew.bat build`)
3. **Find the mod**: The compiled JAR will be in `build/libs/`
4. **Install**: Place the JAR in your NeoForge mods folder
5. **Craft or Find**: The Death Star Block is available in the creative inventory
6. **Trigger**: Right-click the Death Star Block to activate the explosion

## Installation

1. Make sure you have NeoForge 21.0.74+ installed
2. Download the mod JAR from the releases
3. Place it in your `mods` folder
4. Launch Minecraft with the NeoForge profile

## Warning

⚠️ **This mod creates MASSIVE explosions that destroy 1000x1000 blocks!** Use with caution in survival mode or important builds. Test in creative mode first!
