# Milk Splash Potion Mod

A NeoForge mod for Minecraft 1.21.1 that adds splash milk potions!

## Features

- **Splash Potion of Milk**: Throw to remove all potion effects from nearby entities
- **Craftable**: Can be brewed using a brewing stand
- **Area Effect**: Affects all entities within a 4-block radius
- **Works on Players and Mobs**: Functions on both!

## How to Use

1. **Build Requirements**: Java 21 or higher
2. **Build the mod**: Run `./gradlew build` (on Windows: `gradlew.bat build`)
3. **Find the mod**: The compiled JAR will be in `build/libs/`
4. **Install**: Place the JAR in your NeoForge mods folder
5. **Launch**: Start Minecraft with the NeoForge profile
6. **Craft**: Brew a Splash Potion of Milk using a brewing stand
7. **Use**: Throw it at mobs or players to remove their potion effects!

## Recipe

- Combine: **Milk + Potion Base** in a brewing stand to create the splash potion
- Or find it in the creative inventory

## Installation

1. Make sure you have NeoForge 21.0.74+ installed
2. Download the mod JAR from the releases
3. Place it in your `mods` folder
4. Launch Minecraft with the NeoForge profile

## Building from Source

```bash
git clone https://github.com/mrgolden175/Mr-Golden.git
cd Mr-Golden/MilkSplashMod

```
