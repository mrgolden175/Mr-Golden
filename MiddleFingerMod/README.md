# Middle Finger Mod

A fun NeoForge mod for Minecraft 1.21.1 that allows your character to show the middle finger!

## Features

- **Press Z** to make your character show the middle finger
- **3-second duration** - The gesture displays for 3 seconds
- **Client-side only** - No server installation needed for single-player

## How to Use

1. **Build Requirements**: Java 21 or higher
2. **Build the mod**: Run `./gradlew build` (on Windows: `gradlew.bat build`)
3. **Find the mod**: The compiled JAR will be in `build/libs/`
4. **Install**: Place the JAR in your NeoForge mods folder
5. **Launch**: Start Minecraft with the NeoForge profile
6. **Use**: Press **Z** to show the middle finger!

## Keybind

The default keybind is **Z**. You can change this in your Minecraft controls settings:
- Open Settings
- Go to Controls
- Find "Show Middle Finger" and rebind as needed

## Installation

1. Make sure you have NeoForge 21.0.74+ installed
2. Download the mod JAR from the releases
3. Place it in your `mods` folder
4. Launch Minecraft with the NeoForge profile

## Building from Source

```bash
git clone https://github.com/mrgolden175/Mr-Golden.git
cd Mr-Golden/MiddleFingerMod
./gradlew build
```

The compiled JAR will be in `build/libs/`
